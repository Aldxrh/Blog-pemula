HTML untuk Pemula

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web. Dengan HTML, kamu bisa menyusun teks, gambar, link, tabel, form, dan elemen lainnya.

Belajar HTML bisa dimulai dari memahami tag seperti <html>, <head>, <body>, <h1> sampai <a> dan <img>. Semakin sering praktik, semakin cepat kamu paham!
